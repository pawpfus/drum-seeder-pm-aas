# Drum Seeder PM AAS 6:1 — Blueprint 3D

Blueprint 3D interaktif **Drum Seeder** sistem pertanaman **PM AAS (Pertanian Modern · Advanced Agriculture System) 6:1**.

Klik komponen (roda, tabung PVC, tutup paralon, poros, rangka penarik) untuk melihat spesifikasi dimensinya sesuai juknis; komponen terlepas dari rakitan lalu terpasang kembali saat ditutup.

## Fitur
- Model 3D interaktif (Three.js) — putar, zoom, klik komponen.
- Spesifikasi per komponen sesuai juknis (jarak baris 25/50 cm, Ø lubang 1,2 cm, jarak lubang 3 cm, lebar roda 5 cm, PVC 4 inch, rasio 6:1).
- **PWA** — bisa dipasang & dipakai offline.
- Responsif (desktop & mobile).

## Menjalankan
Buka `index.html` lewat server statis (PWA aktif via http/https), mis. GitHub Pages.
